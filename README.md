# CS 510 – Operating Systems

This repository contains coursework, programming assignments, and projects completed for **CS 510 – Operating Systems** as part of the **Master of Science in Computer Science** program at **Southern New Hampshire University (SNHU)**.

Throughout this course, I explored the fundamental concepts of modern operating systems, including process management, CPU scheduling, memory management, concurrency, synchronization, file systems, I/O management, security, and resource allocation. The projects included both theoretical analysis and practical Python implementations that demonstrate how operating systems manage hardware resources while maintaining performance, stability, and security.

# Major Projects

## Project One – Operating System Evaluation

Performed a comprehensive evaluation of **Windows Server 2025** for a fictional financial technology company. The evaluation focused on:

- Security mechanisms
- System reliability
- Performance
- Resource management
- Distributed computing
- Cloud infrastructure
- Enterprise deployment recommendations

The project concluded with a recommendation based on the organization's operational and security requirements.

---

## Project Two – Operating System Performance Simulator

Developed a Python-based operating system simulator that demonstrates several core operating system concepts.

### Features

- Disk usage monitoring
- CPU utilization monitoring
- Memory management statistics
- Multi-threaded execution
- Error and exception handling
- Resource allocation visualization

The application utilizes Python's **psutil** library together with multithreading to simulate and monitor operating system performance metrics.
